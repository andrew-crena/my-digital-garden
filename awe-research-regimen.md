## 🛠️ Week 0 Setup

### System 1: Awe Research Map

Create a spreadsheet with these columns:

| Column | Options/Format |
|--------|----------------|
| **Citation** | APA format |
| **Year** | YYYY |
| **Lab** | Keltner, Piff, Stellar, Gabriel, etc. |
| **Domain** | `nature` / `music` / `visual art` / `film` / `ritual` / `VR` / `recall` |
| **Context** | `individual` / `co-present` / `synchronized collective` |
| **Induction** | Brief description of stimulus/procedure |
| **Awe Measure** | `single-item` / `DPES` / `AWE-S` / other |
| **Comparison** | `neutral` / `amusement` / `admiration` / `beauty` / `elevation` |
| **DV Type** | `behavioral` / `self-report` |
| **Mechanism** | `small self` / `connectedness` / `meaning` / `accommodation` / `humility` |
| **Moderators** | Traits/contexts tested |
| **Validity Flags** | Confounds, weak controls, issues |
| **Verdict** | `A` (must-read) / `B` (useful) / `C` (skip) |
| **Use Tag** | `theory` / `measure` / `method` / `evidence` / `moderator` |
| **One-Line Takeaway** | Your synthesis |
